# AI Search API 연습 프로젝트 제안

## 프로젝트 1: 실시간 뉴스 분석 대시보드 📰

### 개요
사용자가 관심 키워드를 입력하면 Perplexity로 최신 뉴스를 검색하고, Gemini로 요약 및 트렌드 분석을 제공하는 대시보드

### 주요 기능
- **뉴스 검색**: Perplexity API로 실시간 뉴스 검색
- **AI 요약**: Gemini API로 뉴스 내용 요약
- **감성 분석**: 긍정/부정/중립 분석
- **트렌드 시각화**: 키워드 트렌드 그래프
- **북마크 기능**: 관심 뉴스 저장

### 기술 스택
- Next.js 14 (App Router)
- Perplexity API (검색)
- Gemini API (요약/분석)
- Tailwind CSS
- Chart.js (시각화)
- Netlify (배포)

⭐⭐⭐ (중)

## 프로젝트 2: 스마트 리서치 어시스턴트 🔍

### 개요
학생/연구자를 위한 AI 기반 리서치 도구. 질문을 입력하면 Perplexity로 최신 정보를 수집하고, Gemini로 구조화된 리포트를 생성

### 주요 기능
- **질문 기반 검색**: 자연어 질문으로 검색
- **멀티 소스 수집**: Perplexity로 여러 출처에서 정보 수집
- **구조화된 리포트**: Gemini가 서론-본론-결론 형식으로 정리
- **인용 관리**: 자동 출처 표기
- **PDF 내보내기**: 리포트 다운로드
- **검색 히스토리**: 과거 리서치 관리

### 기술 스택
- Next.js 14 (App Router)
- Perplexity API (정보 수집)
- Gemini API (리포트 생성)
- React Markdown
- jsPDF (PDF 생성)
- Netlify Functions
- Netlify Blob Storage (히스토리 저장)

## 프로젝트 3: 여행 계획 AI 플래너 ✈️

### 개요
여행지와 기간을 입력하면 Perplexity로 최신 여행 정보(날씨, 명소, 맛집)를 수집하고, Gemini로 일정표를 생성하는 서비스

### 주요 기능
- **여행지 정보 수집**: Perplexity로 실시간 정보 검색
  - 날씨 정보
  - 인기 관광지
  - 현지 맛집
  - 교통 정보
- **AI 일정 생성**: Gemini로 하루별 상세 일정 생성
- **예산 추천**: 예상 비용 계산
- **지도 연동**: Google Maps 표시
- **일정 수정**: 드래그 앤 드롭으로 일정 조정
- **공유 기능**: 일정 링크 공유

### 기술 스택
- Next.js 14 (App Router)
- Perplexity API (여행 정보 검색)
- Gemini API (일정 생성)
- Google Maps API
- DnD Kit (드래그 앤 드롭)
- Zustand (상태 관리)
- Netlify Functions

## 프로젝트 4: B2B 콜드 마케팅 자동화 SaaS 📧

### 개요
제품 아이디어를 입력하면 Perplexity API로 잠재 B2B 고객을 자동으로 찾고, 이메일/전화번호를 수집하여 콜드 마케팅을 자동화하는 SaaS

### 주요 기능
- **잠재 고객 발굴**: Perplexity로 제품에 맞는 기업/담당자 검색
  - 업종별 타겟팅
  - 기업 규모 필터링
  - 최근 뉴스/자금 조달 정보
- **연락처 수집**: 이메일, 전화번호, LinkedIn 자동 수집
- **리스트 정리**: 자동으로 구조화된 DB 생성
- **AI 이메일 작성**: Gemini로 개인화된 콜드 이메일 생성
- **발송 자동화**: 이메일/메신저 자동 발송
- **응답 추적**: 오픈율, 응답률 대시보드
- **Follow-up 자동화**: 무응답 시 자동 리마인더

### 기술 스택
- Next.js 15 (App Router)
- Perplexity API (리서치)
- Gemini API (이메일 작성)
- SendGrid/Resend (이메일 발송)
- PostgreSQL (고객 DB)
- Prisma (ORM)
- Stripe (결제)
- Netlify/Vercel

### 비즈니스 모델
- Free: 월 100개 리드
- Pro: 월 $49 - 1,000개 리드
- Enterprise: 커스텀 가격

### 법적 고려사항
⚠️ **중요**: GDPR, CAN-SPAM Act, 개인정보보호법 준수 필수

### 난이도
⭐⭐⭐⭐⭐ (상) - 법적 이슈, 데이터 수집 제약

## 프로젝트 5: AI 마트 - 문제 해결 기반 전자상거래 🛒

### 개요
사용자가 문제를 말하면 AI가 질문을 정리하고, Perplexity로 해결책이 될 제품을 찾아 구매 링크를 제공하는 새로운 형태의 쇼핑 경험

### 주요 기능
- **문제 인식**: 사용자가 자연어로 문제 설명
  - 예: "집에서 재택근무 할 때 허리가 아파요"
- **질문 생성**: Gemini가 필요한 질문들을 자동 생성
  - 예: "예산은?", "공간은?", "선호 스타일은?"
- **스마트 검색**: Perplexity API로 제품 검색
  - 리뷰 분석
  - 가격 비교
  - 최신 제품 정보
- **추천 리스트**: AI가 문제 해결에 최적인 제품 큐레이션
- **구매 링크 통합**: 쿠팡, 아마존, 네이버 쇼핑 등 연동
- **비교 분석**: 제품 간 장단점 비교표
- **리뷰 요약**: 실제 구매자 리뷰 AI 요약

### 사용자 플로우
```
1. 문제 입력: "허리 아파요"
   ↓
2. AI 질문: "예산은?", "공간은?"
   ↓
3. 답변 입력
   ↓
4. Perplexity 검색: 인체공학 의자, 스탠딩 책상 등
   ↓
5. 추천 + 구매 링크
   ↓
6. 클릭 → 제휴 수익
```

### 기술 스택
- Next.js 15 (App Router)
- Gemini API (질문 생성, 추천 로직)
- Perplexity API (제품 검색)
- Affiliate APIs (쿠팡 파트너스, 아마존 어소시에이트)
- MongoDB (검색 히스토리)
- Zustand (상태 관리)

### 비즈니스 모델
- 제휴 마케팅 수수료 (3-10%)
- 프리미엄: 광고 제거, 우선 추천

### 차별점
- 기존 쇼핑몰: "제품 검색"
- AI 마트: "문제 해결" → 더 높은 구매 전환율

## 프로젝트 6: AI 기반 경쟁사 모니터링 대시보드 🔎

### 개요
경쟁사 이름을 입력하면 Perplexity로 최신 뉴스, 제품 출시, 채용 정보, SNS 활동을 자동 수집하고 Gemini로 인사이트를 제공하는 비즈니스 인텔리전스 도구

### 주요 기능
- **자동 모니터링**: 경쟁사 실시간 추적
  - 뉴스 기사
  - 제품 출시
  - 채용 공고 (기술 스택 파악)
  - SNS 활동
  - 자금 조달 정보
- **AI 인사이트**: Gemini가 트렌드 분석
  - 전략 변화 감지
  - 위협/기회 분석
  - 시장 포지셔닝
- **알림 시스템**: 중요 변화 시 이메일/슬랙 알림
- **비교 리포트**: 여러 경쟁사 동시 비교
- **시각화**: 타임라인, 차트

### 사용 사례
- 스타트업: 경쟁사 전략 파악
- VC: 투자처 모니터링
- 마케터: 경쟁사 캠페인 분석

### 기술 스택
- Next.js 15
- Perplexity API (정보 수집)
- Gemini API (분석)
- Cron Jobs (정기 수집)
- PostgreSQL
- Chart.js

### 비즈니스 모델
- Free: 1개 경쟁사
- Pro: $29/월 - 5개
- Team: $99/월 - 무제한

⭐⭐⭐⭐ (중상)
